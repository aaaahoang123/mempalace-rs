# Security Policy

## Supported Versions

Only the latest release of `mempalace-rs` is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of `mempalace-rs` seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not** open a public issue.
2. Send a detailed report to the maintainers (via GitHub private vulnerability reporting or direct contact).
3. Include a description of the vulnerability, steps to reproduce, and potential impact.

We will acknowledge receipt of your report within 48 hours and provide a timeline for resolution if the vulnerability is confirmed.
